Este es un sitio web del clima que hice utilizando el API de Openweather "https://openweathermap.org/api", es un sitio responsive y minimalista el cual mediante un busca
dor el cual se encuentra centrado en el medio de la pantalla recibe la informacion de la ciudad/pais/localidad que ponga el usuario, en el javascript mas alla de la 
promesa que utilizo para lograr sacar la informacion del API pueden ver que hay muchas condiconales, estas condicionales son las que hacen que el sitio web detecte
cuando hace calor, frio o esta templado. Cualquier duda sobre mi codigo sientanse libre en contactarme y les respondere a la brevedad
